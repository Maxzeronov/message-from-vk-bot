# message-from-vk-bot v0.1
## Программа для отправки сообщения в VK, от имени бота(сообщества) на Python.

---

## Как запустить программу?
__1. Установить нужные модули:__
```
pip install vk_api
pip install PyQt5
```
__2. Запустить python файл:__
```
python gui.py
```
